# Expense Tracker & Productivity Suite
A personal dashboard for managing daily expenses and task tracking, built with Python and Flet.

## Features
- **Dashboard**: Real-time expense metrics and charts.
- **Expenses**: Log, edit, and categorize your daily spending.
- **Pomodoro**: Integrated focus timer.

## Setup Instructions
1. Clone this repository: `git clone <github.com/Unavailable4u/FocusOS>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python main.py`

## Distribution
To create a standalone executable for Windows, run:
`flet build windows`
